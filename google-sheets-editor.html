⚙️ Google API 설정
OAuth 2.0 클라이언트 ID를 입력하면 브라우저에 저장됩니다.
📋 빠른 설정 가이드
1. Google Cloud Console → 새 프로젝트 생성
2. Google Drive API + Google Sheets API 활성화
3. 사용자 인증 정보 → OAuth 2.0 클라이언트 ID 생성 (웹 애플리케이션)
4. 승인된 JavaScript 원본에 http://localhost 추가
5. 클라이언트 ID 복사 → 아래 입력 → 저장
